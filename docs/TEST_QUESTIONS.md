# RECOMENDACIONES DE IMPLEMENTACIÓN

## 1. Validación del Instrumento

Antes de lanzar, considera hacer un **piloto con 50-100 estudiantes** de diferentes semestres y comparar resultados con sus preferencias declaradas. Esto te permitirá ajustar ponderaciones.

## 2. Prevención de Respuestas Socialmente Deseables

Algunas preguntas (especialmente las éticas) pueden generar respuestas "correctas". Considera:

- Añadir preguntas de verificación/consistencia
- Usar escalas de deseabilidad social como control
- Mezclar el orden de preguntas

## 3. Adaptación al Semestre del Estudiante

Los de primeros semestres pueden no entender ciertos términos técnicos. Considera:

- **Versión simplificada** para 1°-2° semestre
- **Versión completa** para 3°+ semestre

## 4. Feedback Contextualizado

Para cada perfil recomendado, incluye:

- ✅ Materias clave de la carrera
- ✅ Áreas de práctica profesional en México
- ✅ Perfil típico de empleadores
- ✅ Rango salarial inicial (con fuentes como IMCO o encuestas de universidades)
